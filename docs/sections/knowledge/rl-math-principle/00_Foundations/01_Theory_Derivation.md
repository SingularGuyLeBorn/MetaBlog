## 1. 从直觉到严格定义：为什么需要概率空间？ 

### 1.1 大学概率论的处理方式 

在大学概率论课程中，我们通常这样定义概率：

例子：投掷一枚均匀硬币

- 可能的结果：正面(H) 或 反面(T)
- 每个结果的概率：正面反面P(正面)=P(反面)=0.5

这种处理方式对于离散的、有限的情况是足够的。

## 3. 随机变量：从样本空间到实数的桥梁 

### 3.1 随机变量的定义 

定义：随机变量 X 是一个函数，它将样本空间 Ω 中的每个样本点 ω 映射到一个实数。

形式化表示：

X:Ω→R符号解释：

Ω：样本空间（所有可能结果的集合）R：实数集（…,−2,−1,0,1,2,… 以及所有小数）X(ω)：当实验结果为 ω 时，随机变量取的值

### 3.2 为什么随机变量是"函数"而不是"变量"？ 

这是初学者最容易困惑的地方。让我们用例子来说明。

例子：投掷两枚硬币

样本空间：Ω={(H,H),(H,T),(T,H),(T,T)}

定义随机变量 X = "正面朝上的次数"

那么 X 作为函数的作用是：

样本点ωX(ω) 的值(H,H)2(H,T)1(T,H)1(T,T)0所以，X 并不是一个固定的数值，而是一条"规则"——告诉我们如何从每个实验结果得到一个数字。

图2：随机变量作为映射函数

图注：左边是样本空间 Ω，包含4个样本点 ω1,ω2,ω3,ω4。右边是实数轴 R。蓝色箭头表示随机变量 X 将每个样本点映射到一个实数。注意 ω2 和 ω3 都被映射到了同一个值1。

### 3.3 随机变量的"可测性"条件 

随机变量不能是任意的函数，它需要满足一个技术条件：可测性 (Measurability)。

可测性的含义：对于实数轴上的任何"合理的"集合 B（例如区间 (−∞,3]），我们都能追溯回去找到 Ω 中对应的事件。

形式化表达：

{ω∈Ω:X(ω)∈B}∈F逐项解释：

B：实数轴上的一个集合，例如 B=(−∞,3] 表示"所有小于等于3的实数"X(ω)∈B：随机变量的值落在集合 B 内{ω∈Ω:X(ω)∈B}：使得 X 的值落在 B 内的所有样本点组成的集合这个集合必须属于 F，也就是说它必须是一个"可以讨论概率"的事件为什么需要这个条件？

因为我们要能够计算 P(X≤3) 这样的概率。这个概率实际上是：

P(X≤3)=P({ω:X(ω)≤3})如果右边的集合不在 F 中，我们就无法用 P 来计算它的概率！

## 4. 期望：从求和到积分 

### 4.1 离散情况下的期望 

对于离散随机变量 X，其取值为 x1,x2,…，期望定义为：

E[X]=∑ixi⋅P(X=xi)逐项解释：

xi：随机变量可能取到的某个值P(X=xi)：取到这个值的概率求和遍历所有可能的值整体含义：用概率加权的平均值例子：投掷一枚均匀骰子，X = 点数

E[X]=1⋅16+2⋅16+…+6⋅16=3.5

### 4.2 连续情况下的期望 

对于连续随机变量 X，如果它的概率密度函数是 p(x)，期望定义为：

E[X]=∫−∞∞x⋅p(x)dx逐项解释：

p(x)：概率密度函数。注意：p(x) 本身不是概率，而是概率的"密度"。p(x)dx 才近似表示 X 落在 [x,x+dx] 区间的概率。∫−∞∞：对整个实数轴积分，对应于"遍历所有可能值"积分替代了求和，因为可能的值是连续的、不可数的

### 4.3 统一视角：勒贝格积分 (Lebesgue Integral) 

测度论提供了一个统一的框架，将离散和连续情况都纳入同一个公式：

E[X]=∫ΩX(ω)dP(ω)逐项解释：

∫Ω：在整个样本空间 Ω 上积分X(ω)：随机变量在样本点 ω 处的取值dP(ω)：用概率测度 P 来"加权"整体含义：对所有可能的结果，用其概率加权求和（或积分）这个公式为什么重要？

统一性：无论 Ω 是离散的还是连续的，公式形式相同。严格性：它基于测度论，可以处理复杂的概率分布（例如混合分布）。理论基础：强化学习中的价值函数 V(s)=E[Gt|St=s] 正是这种条件期望。
